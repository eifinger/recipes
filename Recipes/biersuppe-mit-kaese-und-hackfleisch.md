# Biersuppe mit Käse und Hackfleisch

> Original: [Chefkoch Magazin](https://www.chefkoch.de/magazin/artikel/114795983823289-biersuppe-mit-kaese-und-hackfleisch-1)

## Zutaten (2 Portionen)

| Menge | Zutat |
|-------|-------|
| 0.5 | Brezel |
| 0.5 | Zwiebel |
| 1 Schuss | Pflanzenöl |
| 200 g | Hackfleisch |
| 1 Zehe | Knoblauch |
| 400 ml | Bier |
| 350 ml | Brühe |
| 200 ml | Sahne |
| 200 g | Emmentaler, gerieben |
| 200 g | Cheddar, gerieben |
| 1 Prise | Paprikapulver |
| | Salz & Pfeffer |
| | Schnittlauch (zum Dekorieren) |

## Zubereitung

### Topping

1. Die Brezel in Scheiben schneiden. In etwas Öl mit Zwiebeln knusprig braten.

### Suppe

2. Das Hackfleisch mit einer abgezogenen und gehackten Knoblauchzehe krümelig braten.

3. Mit Bier und Gemüsebrühe ablöschen.

4. Sahne hinzufügen sowie Emmentaler und Cheddar einrühren und schmelzen lassen.

5. Alles mit Paprikapulver, Salz und Pfeffer würzen und köcheln lassen, bis die Suppe eine cremige Textur hat.

6. Optional mit etwas angerührter Stärke abbinden.

7. Mit den Brezeln und etwas Schnittlauch servieren.

## Tipps

- **Bierwahl:** Im Original ein mildes Bier verwendet – aber auch Rauchbier oder dunkles Bier geben eine besondere Note.
- **Hackfleisch:** Gemischtes Hack funktioniert, aber Rinderhack oder Geflügelhack sind auch Varianten.
- **Optional:** Mit angerührter Stärke abbinden für extra Cremigkeit.
